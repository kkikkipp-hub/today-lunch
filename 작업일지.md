### 기능 — 카카오 로컬 API 연동
- Kakao Local API로 위치 기반 주변 맛집 검색 구현
- 앱 진입 시 자동 위치 권한 요청

- Toss 미니앱 심사 요구사항 반영 (개인정보처리방침, 이용약관, 고객지원 페이지)

### 기능 — 바텀시트 식사 기록
- 추천 메뉴 카드 → 근처 가게 바텀시트 (카카오 메뉴명 검색)
- 가게 선택 시 인라인 가격 입력 필드 표시
- 가격 직접 입력 후 "기록하기" → localStorage 저장
- 바텀시트 에러 메시지 상세 표시

---

### 기능 7개 — 사용자 니즈 리서치 기반 추가

직장인 점심 앱 사용자 니즈 분석 후 7개 기능 순차 구현:

| # | 기능 | 기술 |
|---|------|------|
| 1 | 이번 주 먹은 메뉴 자동 제외 | localStorage 기록 기반 필터링 |
| 2 | 날씨 연동 추천 | Open-Meteo API (무료, 키 없음) / 기온·강수 기반 무드 가중치 |
| 3 | 추천 결과 공유 | Web Share API + 클립보드 복사 fallback |
| 4 | 단골 가게 즐겨찾기 | 하트 버튼, localStorage, 바텀시트 상단 정렬 |
| 5 | 점심 알림 배너 | 11~14시 사이 오늘 기록 없으면 인앱 배너 표시 |
| 6 | 월간 리포트 | 히스토리 탭 주간/월간 분리, SVG 바차트, 기록 90일 보존 |
| 7 | 영수증 OCR | 카메라 촬영 → ocr.space API → 금액 자동 인식 후 입력 |
